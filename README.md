# SimpleNewsApp
------------
## Usage
locale.properties에
[NewsAPI.org](https://newsapi.org/) 에서 발급받은 API KEY를 추가한 후 사용합니다

NEWS_API_KEY=your_api_key

## 사용한 것들
- Kotlin
- Retrofit2
- RxJava2, RxKotlin
- Glide 4.8
